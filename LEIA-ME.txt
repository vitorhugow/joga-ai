COMO APLICAR (escolhe UMA das opções):

OPÇÃO A — Copiar ficheiros (mais simples):
  Extrai este zip na RAIZ do repositório (a pasta que contém
  package.json e artifacts/). Os ficheiros substituem os antigos
  nos caminhos corretos. Confirma a substituição quando perguntado.

OPÇÃO B — Aplicar o patch git (só ficheiros de texto):
  git apply joga-ai-fixes.patch
  (as imagens novas/otimizadas vêm no zip de qualquer forma)

DEPOIS DE APLICAR:
  1. cd artifacts/joga-ai && pnpm install && pnpm run build  (deve passar)
  2. Revê o diff das firestore.rules com calma
  3. Deploy das regras: firebase deploy --only firestore:rules
  4. Testa manualmente: entrar numa comunidade pública, aprovar
     joinRequest numa privada, registar golos numa partida ao vivo
     (como organizador) e apagar uma partida finalizada (revert).
  5. git commit + push → Cloudflare Pages faz o deploy do site.
